# Portfolio Putra Ade Akbar

Website portfolio statis dengan gaya editorial ringan, terinspirasi oleh ritme layout `kuncoro.io` tanpa menyalin tampilannya secara literal.

## File utama

- `index.html`: struktur konten utama
- `styles.css`: tampilan visual dan responsif
- `script.js`: animasi reveal ringan, highlight navigasi, dan tahun footer

## Cara edit cepat

- Ubah copy tulisan langsung di `index.html`
- Ubah warna, font, dan spacing di bagian `:root` pada `styles.css`
- Jika ingin menambah email, GitHub, atau CV, tambahkan link baru di section `#kontak` dan card hero

## Catatan

Profil LinkedIn publik tidak bisa diambil otomatis dari environment ini, jadi konten detail pengalaman belum disinkronkan secara otomatis dan disiapkan dalam format yang mudah Anda lanjutkan.
